# BitSync – Real-Time Chat Application 💬

BitSync is a full-stack real-time chat application that allows users to send messages instantly, create groups, and share files. It is built to understand how real-time communication works in modern web applications.

---

## 🚀 Live Demo
https://bit-sync-chat-app.vercel.app

---

## ✨ Features

- 🔹 Real-time messaging using Socket.IO  
- 🔹 One-to-one and group chat support  
- 🔹 Secure user authentication using JWT  
- 🔹 File sharing (images and documents)  
- 🔹 Responsive UI for all screen sizes  
- 🔹 Fast message delivery with low latency  

---

## 🛠️ Tech Stack

**Frontend**
- React.js  
- Tailwind CSS  

**Backend**
- Node.js  
- Express.js  

**Database**
- MongoDB Atlas  

**Other Tools**
- Socket.IO (real-time communication)  
- JWT (authentication)  

---

## ⚙️ How It Works

- Users can register and log in securely  
- Messages are sent instantly using WebSockets (Socket.IO)  
- Backend APIs handle users, chats, and messages  
- Data is stored in MongoDB  
- Files can be shared directly in chat  

---

## 🧠 What I Learned

- How real-time communication works using Socket.IO  
- Handling authentication and user sessions with JWT  
- Building REST APIs using Node.js and Express  
- Managing frontend state and UI in React  
- Connecting frontend and backend in a full-stack project
